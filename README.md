# 🚀 Element | Floating

## 🛠️ Tecnologie Utilizzate

- **HTML5:** Struttura semantica avanzata (`<header>`, `<main>`, `<section>`, `<picture>`, `<aside>`).
- **CSS3:** Stile personalizzato, layout flessibili, animazioni di dissolvenza (`fade`) e transizioni fluide.
- **JavaScript (Vanilla JS):** Logica di controllo per slider, barra di navigazione, gestione form e pop-up dei cookie, senza dipendenze esterne.
- **Font Awesome (v5.15.3):** Icone social caricate in modo asincrono per evitare il blocco del rendering.
- **Animate.css:** Animazione floating di due elementi
